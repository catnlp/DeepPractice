# DeepPractice

Deep practice for deep learning

项目分为理论和实践，因为最近在找工作，所以先完成理论部分，再实践部分

理论部分主要是吴恩达老师的视频课程和参考了网上的优秀博客

实践部分打算在PyTorch框架下完成，从最基本的模块实现相应功能

## 1 Environment

Python: 3.6.6

PyTorch: 0.4.1

```
$ conda install pytorch torchvision jupyter matplotlib
$ pip install visdom
```
